Project 1: Intro to JavaScript statements, variables, the Console, and the console.log() statement

Instructions:

  1. Create a semantically correct, validated, and spell checked HTML web page with the following questions, answers, and exercises.
  2. Add a pleasing external style sheet. No, I'm not grading in your styles but practicing CSS is important as we will use JavaScript to modify styles soon too!
  3. Add your name and the current date at the top of every file as a comment.
  4. Publish the completed web page to the student web server and post the URL to your web page here.

Questions

  1. What is the purpose of JavaScript? Answer in general and also provide three specific examples of how JavaScript is commonly used.
  2. Who invented JavaScript and when?
  3. What software program will you be using to write JavaScript?
  4. What software program will you be using to run JavaScript?
  5. Why are external scripts preferable to scripts added directly within an HTML file?

Exercises

  1. Add a script that alerts "Hi World!"
  2. In the script created in step 1, add a statement to console.log("Hi Insert Your Name");
  3. In the script created in step 1 and also step 4, include the "use strict": directive as the first line.
  4. Add a second script that declares 2 variables: a constant named myName assigned to your actual name as a string, and a variable named mySalary assigned to any number that you would like as a salary. Use console.log and the variable names to log out a phrase similar to the following: My name is Lisa and my salary is 100000. Note that you must include text in quotation marks (or apostrophes), but not variable names similar to the following: console.log("My name is " + myName + " and my salary is " + mySalary);
  5. In the script you created in step 4, but below the first console.log, reassign the mySalary variable to a 150000. Add another console.log similar to the following: My name is Lisa and my new salary is 150000
  6. In the script you created in step 4, create a Boolean variable named veteran and assign it to true or false depending on whether you are a veteran or not.
  7. In the script you created in step 4, add three console.log() statements to determine the data type of the myName, mySalary, and veteran variables. An example is: console.log(typeof(myName));
  8. In the script you created in step 4, console.log each of these expressions (they are found at the end of the Type Conversions lesson) Make sure you understand these! The + operator is the tricky one given it will NOT convert a string.
    • "" + 1 + 0
    • "" - 1 + 0
    • true + false
    • 6 / "3"
    • "2" * "3"
    • 4 + 5 + "px"
    • "$" + 4 + 5
    • "4" - 2
    • "4px" - 2
    • 7 / 0
    • " -9 " + 5
    • " -9 " - 5
    • null + 1
    • undefined + 1
  9. In the script you created in step 4, add the following statements.

    • Add a comment to explain the value of d.

  10. In the script you created in step 4, add the following statements. In two different statements after these variable declarations, console.log the value of x and y.
