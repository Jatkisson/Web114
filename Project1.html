Questions

What is the purpose of JavaScript? Answer in general and also provide three specific examples of how JavaScript is commonly used.
The purpose of JavaScript is to make a web page more interactive. HTML gives the structure, CSS gives the style, and JavaScript gives the page movement and life. For example, Javascript can add new HTML to a page, it can provide reactions to use actions within the page, and it can get and set cookies to provide a unique experience for the end user.
Who invented JavaScript and when?
JavaScript was invented by Brendan Eich in 1995.
What software program will you be using to write JavaScript?
I will be using Visual Studio Code as the program to write JavaScript for this course as I am somewhat familiar with the program and am comfortable with using the software.
What software program will you be using to run JavaScript?
I will be using the Google Chrome browser to run Javascript. This web browser uses the V8 Javascript engine to execute the language.
Why are external scripts preferable to scripts added directly within an HTML file?
External scripts are preferred when developing web pages as the script can be reused on multiple pages and can be updated in one location instead of updating the script on each individual HTML page.
