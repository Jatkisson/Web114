Project 4: Creating variables and using alert(), confirm(), and prompt() to interact with the user, using if-then-else structures

Instructions:

  1. Create a semantically correct, validated, and spell checked HTML web page with the following questions, answers, and exercises.
  2. Add a pleasing external style sheet. No, I'm not grading in your styles but practicing CSS is important as we will use JavaScript to modify styles soon too!
  3. Add your name and the current date at the top of every file as a comment.
  4. Publish the completed web page to the student web server and post the URL to your web page here.

Questions

1. alert(), confirm(), and prompt() are methods of which object?
alert(), confirm(), and prompt() are methods of the default object / Window object that allows developers to interact with the user
2. Why can we use the alert(), confirm(), and prompt() methods without preceding them with the object. (object dot) syntax??
The alert(), confirm(), and prompt() are able to be used without the preceding object dot notation because it is the default global object does not require calling the object, but the alert(), confirm(), and prompt() methods will still work the same if the window object dot notation is used.

Exercises

  1. Add a statement that declares a variable named user and assign the variable to the value of the answer given to a JavaScript prompt. The prompt should ask the user for his or her name.
  2. Add a statement that declares a variable named proceed and assign the variable to the value of the answer given to a JavaScript confirm. The text should ask the user permission to use his or her name on the web page.
  3. Write a traditional if - else statement that assigns the value in the user variable as the textContent property of an <h1> element IF the value of the proceed variable is true. If the proceed variable is false, the textContent property of the <h1> element should say "Privacy Please".
  4. Rewrite the if - else statement in #3 using the conditional ? operator.
  5. Prompt the user for his or her age, and store the result in a variable named age.
  6. Add a window.alert() statement that alerts out the person's name and age as a final statement in your JavaScript.
